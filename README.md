# 오목만들기 프로젝트

## 1. 동기
  + 쉽게 즐길수있는 게임을 만들고자 오목을 제작을 진행함.
## 2. 규칙
  + 각 두명의 플레이어가 게임을 진행함
  + 5개의 같은 바둑돌이 가로 세로 대각선 으로 나오면 승리
  + 30초 동안 돌을 두지않을경우 다른플레이어에게 넘어간다.
